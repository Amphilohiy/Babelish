Babelish

Chaotically confused, like Babel

View project onGitHub

Babelish is a command tool to convert a csv file of translations into localisation files such as:

  • .strings (iOS)
  • .xml (Android)
  • .json
  • .php

Requirements

  • Ruby 1.9.2 or above

Setup

gem install babelish

Usage

babelish help

  • Use configuration file

You can use a configuration file to hold all your commandline arguments into a file. Place a .babelish file in your repo where you will run the command. See .babelish.sample as the possible values.
